# Capstone_1: Sales Territory Analysis - New York Territory

## Project Description
For this capstone I am demonstrating my ability to read data and extract useful 
information using SQL and create visualization with Excel.

In the scenario for the capstone I have started a job as a data analyst for EmporiUm. 
A sales manager named SEE Ellefson has asked me to analyze their sales territory which is New York. 
I wrote a .sql script to find the answers to the managers questions. 
Then exported the results to a .csv and used that to create Pivot Charts in Excel.

## Territory Details
- Sales Manager: See Ellefson
- Region: East
- Regional director Ana Jilani
- Territory: New York State
- Data Period: January 1, 2022 - Decemer31, 2025

## Key findings
- Total territory revenue: $4,330,817
- Top performing store: New York City
- Bottom performing stores: Queens and Oswego
- Lowest revenue month: May (5)
- Lowest average transactions category: Stationary & Supplies
